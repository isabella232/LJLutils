What it is:

LJLutils is a Java Class Library that is supposed to help developers developing faster by having methods like visualizing tree-structures in the console by automatically indenting them and so on. It will be continually extended as there are more things developed by the author, which he thinks are worthy of putting into this library.

Topics already adressed by this library:

  • Console Printing
  • Physics and Vector Calculations (WIP)
You can find the instructions on how to use this library on the Github page of the project.
